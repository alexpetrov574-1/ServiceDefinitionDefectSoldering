# Сервис классификации дефектов пайки

## Краткое описание задачи

Сервис предназначен для автоматической классификации дефектов пайки радиоэлементов на печатных платах.
Поддерживается 5 классов:
- **normal** - качественная пайка
- **unsolder** - недостаток припоя
- **cold** - холодная пайка
- **excess** - избыток припоя
- **bridge** - перемычка припоя

## Архитектура решения

- **Модель**: ResNet18 с дообучением (transfer learning)
- **Препроцессинг**: Resize до 224×224, нормализация (ImageNet stats)
- **Сервис**: FastAPI + Uvicorn
- **Формат входных данных**: JPG, JPEG, PNG (макс. 10 МБ, мин. 64×64 пикселей)
